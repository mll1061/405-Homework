Vocab Definitions

  • HTTP
    • Transfer of hypermedia documents on the Web, usually between browser and server, so people can read them.
    • Link
  • HTML
    • Specifies webpage structure.
    • Link
  • Element
    • Part of a webpage. May contain a data item, chunk of text, an image, or nothing at all. Usually includes an opening tag with some attributes, enclosed text content, and a closing tag.
    • Link
  • Tag
    • Used for creating elements.
    • Link
  • Block (CSS)
    • An element appearing on a new line. Blocks are also affected by positioning schemes and position property.
    • Link
  • Doctype
    • Prevents a browser from switching into “quirks mode”, ensures that the browser makes a “best-effort” to follow specifications.
    • Link
  • Head
    • Part of an HTML document containing metadata about the document (i.e. author, description, link to files applied to the HTML).
    • Link
  • Attribute
    • Extends a tag, changing its behavior or providing metadata.
    • Link
  • Boolean
    • Can only have the values true or false.
    • Link
  • DOM
    • A document model in the browser representing the document as a node tree, each node representing parts of the document.
    • Link
  • CSS
    • Controls how webpages look in a browser.
    • Link
  • Property (CSS)
    • A characteristic (like color) whose associated value defines one aspect of how the browser should display the element.
    • Link
  • Selector (CSS)
    • The part of a CSS rule that describes what elements in a document the rule will match. The matching elements will have the rule's specified style applied to them.
    • Link
